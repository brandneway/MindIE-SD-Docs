# 使用 Dockerfile 构建开发镜像

MindIE-SD 提供了基于Atlas 800I A2 推理服务器的AArch64环境的Dockerfile文件[docker/Dockerfile_910b_aarch64.ubuntu](https://gitcode.com/Ascend/MindIE-SD/blob/dev/docker/Dockerfile_910b_aarch64.ubuntu)。快速构建开发镜像和容器，详情请参见[MindIE SD Dockerfile 使用指导](../user_guide/install/source/dockerfile_instruction.md)。

# 使用 Dev Container 构建开发容器

MindIE-SD 提供了基于Vs Code Dev Container构建标准化容器的配置，以便于个人PC快速配置开发环境。详情请参见[MindIE-SD DevContainer 分步指南](../user_guide/install/source/devcontainer_instruction.md)。

# 从零构建开发环境

如果上述两种方式都不符合您的需求，也可以选择从零开始。详情请参见[环境准备](../user_guide/install/source/environment_preparation.md)和[源码编译安装](../user_guide/install/source/installation_from_source_code.md)。
