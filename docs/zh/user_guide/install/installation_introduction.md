# 安装说明

MindIE SD是一个Python包，它可以基于PyTorch构建，独立安装、集成到Python应用程序中。此外，MindIE SD是MindIE软件的高性能视图生成推理组件，安装MindIE软件即可获取MindIE SD，详情请参见[MindIE安装说明](https://www.hiascend.com/document/detail/zh/mindie/230/envdeployment/instg/mindie_instg_0001.html)。本安装指南将会介绍如何安装、升级和卸载MindIE SD，使用Dockerfile构建Docker镜像和使用Dev Container构建开发容器。

## 快速构建Docker开发环境

**本文档包含MindIE-SD快速构建Docker开发环境的方案**，包含使用 Dockerfile 构建开发镜像和使用 Dev Container 构建开发容器两种方式。用户可根据自身需求选择使用哪种方式。

- 长期使用MindIE-SD进行模型开发、推理运行等，推荐使用 Dockerfile 构建开发镜像
- 个人PC构建开发环境、快速运行MindIE-SD的测试用例或者其他临时性的开发，推荐使用 Dev Container 构建开发容器
